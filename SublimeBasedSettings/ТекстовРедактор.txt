Вътрешната мотивация

Ctrl + P -> намира файл 
Ctrl + N -> отваряме нов файл
Ctrl + C ->  прекъсва изпълнението на програмата
Ctrl + A -> маркира целият код и прави провера къде имаме чертички и точки, 
избягаме точките. Натискането на таба ни прави чертичка.
Ctrl + Shift + P -> Командното меню на Subline Convert to Space -> 

dir -> листва всички файлове в текущата директория

Конфигурационен файл. Preferences -> Settings -> Syntax Specific

Indentation error -> Играе само табулацията, а не space

Preferences -> Colors Scheme

Пишейки python в cmd, отиваме в Shell.
